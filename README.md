# Student Marks Predictor

This project predicts student marks based on study hours using Machine Learning.

## Tech Stack

- FastAPI
- Scikit-learn
- HTML, CSS, JavaScript
- Render (Deployment)

## Features
- Predict marks using ML model
- API built with FastAPI
- Frontend UI for user input
- Live deployment
